--- title: "Software" permalink: /software author_profile: true --- None of these are "release" quality and they are not actively being maintained, but I may push updates every so often. Please let me know if you find any of these useful, or if you have any questions or comments!
  • SaturnLs: a command line tool to convert Saturn Ls and date.
  • MCSPy: a python package to download and access atmospheric profiles from the Mars Climate Sounder.
  • contourdyn: a vorticity contour dynamics script based on a code by Darryn Waugh.
